# Jogo da Forca (Hangman) em Python

Implementação do clássico **Jogo da Forca** utilizando Python. O jogo é executado no terminal, onde o jogador deve descobrir uma palavra secreta letra por letra antes que o desenho da forca seja completado.

---

## Objetivo do Projeto

Este projeto foi desenvolvido para praticar conceitos fundamentais de programação em Python, como:

- Lógica de programação
- Estruturas de repetição
- Estruturas condicionais
- Manipulação de strings
- Controle de estado do jogo
- Utilização de conjuntos (`set`)
- Geração de valores aleatórios

---

## Funcionalidades

- Sorteio aleatório de palavras
- Desenho da forca atualizado a cada erro
- Validação de entradas inválidas
- Controle de letras já utilizadas
- Contagem de tentativas restantes
- Mensagem de vitória ou derrota
- Opção de jogar novamente

---

## Tecnologias Utilizadas

- Python 3
- Biblioteca nativa `random`

---

## Como Executar

1. Clone este repositório:

```bash
git clone https://github.com/seu-usuario/jogo-da-forca.git
```

2. Acesse a pasta do projeto:

```bash
cd jogo-da-forca
```

3. Execute o jogo:

```bash
python jogo_forca.py
```
